Beneficios

Visión

Cómo funciona

La plataforma integra un frontend responsivo con un backend Node.js que almacena usuarios y gestiona autenticación.

Frontend moderno

Formularios controlados, validación en tiempo real y almacenamiento seguro del token en navegador.

Backend robusto

API Express con endpoints de registro, login y dashboard. Usuarios persistidos en JSON con contraseñas cifradas.

Roles diferenciados

Admin recibe métricas del sistema, Coach gestiona atletas, User monitorea su progreso personal.

Despliegue fácil

Ejecuta `npm install` y `npm start` en el backend. El frontend se conecta automáticamente a `http://localhost:4000`.
